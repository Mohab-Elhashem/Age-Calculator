# ⏳ Age Calculator

A sleek, accurate, and responsive web application that calculates a user's exact age in years, months, and days based on their birthdate. 

## 🚀 Live Demo
You can check out the live project here: [View Live Demo](https://mohab-elhashem.github.io/Age-Calculator/)

## ✨ Features
* 📅 **Accurate Calculation:** Computes age precisely down to the exact number of years, months, and days.
* 📱 **Responsive Design:** Fully optimized for all screen sizes, from mobile phones to desktop monitors.
* 🛑 **Input Validation:** Prevents users from selecting future dates or leaving inputs empty.
* 🎨 **Clean UI:** Modern, user-friendly interface with smooth transitions.

---

## 🛠️ Built With
* **HTML5:** For the semantic structure of the web page.
* **CSS3:** For modern styling, layouts (Flexbox/Grid), and responsiveness.
* **JavaScript (ES6):** For handling the date logic, calculations, and DOM manipulation.
